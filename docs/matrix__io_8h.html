Detailed Description

Stream output operators for all matrix types in the trackingLib math library.

This file provides template-based operator<< implementations for streaming matrix objects to any std::ostream (cout, files, stringstream). The design uses SFINAE-based type detection to work with all matrix-like types that provide the required interface.

Key features:

  • Works with any std::ostream (cout, files, stringstream)
  • Template-based implementation for compile-time type safety
  • SFINAE-based type detection for C++17 compatibility
  • Specialized formatting for DiagonalMatrix, TriangularMatrix, and Vector
  • Future-proof design using at_unsafe() interface
Note
This file replaces the previous print() methods across all matrix types, providing idiomatic C++ stream output without code duplication.
