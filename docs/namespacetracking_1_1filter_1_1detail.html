Decorrelate a measurement for sequential (scalar) processing.

Sequential measurement updates process one measurement component at a time and therefore require uncorrelated components. Using the UDU factorization R = U*D*U' the transformed system vec' = inv(U)*vec ; H' = inv(U)*H has the exactly diagonal measurement covariance R' = D, so the scalar updates on the transformed system are equivalent to a block update with the correlated R.

  • Already diagonal R: no transformation, R' = diag(R).
  • Factored policy: U and D are stored directly — decorrelation needs no decomposition.
  • Full policy with correlations: R.decomposeUDUT() provides U and D.
