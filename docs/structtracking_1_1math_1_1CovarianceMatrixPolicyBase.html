Detailed Description

Base class for covariance matrix policy types.

This empty base class serves as a common ancestor for all covariance matrix policy types. It enables type checking and provides a common interface for policy-based design patterns.

Policy classes derived from this base must provide:

  • A value_type type alias defining the floating-point type
  • A template alias template <sint32 Size> using Instantiate for creating covariance matrix instances
