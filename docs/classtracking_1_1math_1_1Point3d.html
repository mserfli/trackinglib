Detailed Description

template<typename ValueType_>
class tracking::math::Point3d< ValueType_ >

3D point with named coordinate access

Point3d represents a point in 3D space with x, y, and z coordinates. It inherits from Vector<ValueType_, 3> and provides convenient named accessors for the coordinates, making geometric code more readable.

All Vector operations are available, plus coordinate-specific accessors.

Template Parameters
ValueType_The atomic data type of internal elements

Example usage:

Point3d<double> p = Point3d<double>::FromValues(1.0, 2.0, 3.0);
double x = p.x(); // 1.0
double y = p.y(); // 2.0
double z = p.z(); // 3.0
p.z() = 4.0; // modify z coordinate
