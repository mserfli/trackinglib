Detailed Description

Error types used throughout the math library.

This file defines the error enumeration used by the math library's error handling system. These errors are returned via tl::expected<T, Errors> to provide safe error propagation without exceptions.

Note
All error values are used in conjunction with tl::expected to indicate operation failures in matrix and vector operations.
