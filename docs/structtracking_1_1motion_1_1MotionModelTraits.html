Detailed Description

template<typename CovarianceMatrixPolicy_, typename StateDef_>
struct tracking::motion::MotionModelTraits< CovarianceMatrixPolicy_, StateDef_ >

Traits structure for motion model type information.

This template struct provides type information and constants for motion models. It serves as a central location for accessing common type definitions and dimensional information that are used throughout the motion model hierarchy.

The traits structure is used by motion models to:

  • Define the covariance matrix policy type
  • Extract the floating-point type from the policy
  • Access the state definition structure
  • Provide the state dimension as a compile-time constant
Template Parameters
CovarianceMatrixPolicy_Policy type defining the covariance matrix implementation
StateDef_State definition structure containing state variable enumeration

Usage example:

// Define traits for a motion model
using MyTraits = MotionModelTraits<FullCovarianceMatrixPolicy<float32>, StateDefCV>;
// Access type information
using value_type = MyTraits::value_type; // float32
constexpr sint32 stateSize = MyTraits::Size; // StateDefCV::NUM_STATE_VARIABLES
