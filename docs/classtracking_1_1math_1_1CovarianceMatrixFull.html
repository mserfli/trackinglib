Detailed Description

template<typename ValueType_, sint32 Size_>
class tracking::math::CovarianceMatrixFull< ValueType_, Size_ >

Full covariance matrix representation for Kalman filtering.

A covariance matrix represents the uncertainty of a random vector in Kalman filtering. This class provides a standard full matrix representation where all elements are stored. Covariance matrixes are always symmetric and positive semi-definite.

Key properties:

  • Symmetric: P[i,j] = P[j,i] for all i,j
  • Positive semi-definite: x^T*P*x >= 0 for all vectors x
  • Represents uncertainty in state estimation

This implementation uses row-major storage for consistency with other matrix types. All constructors enforce symmetry through assertions.

Set diagonal variance element and clear correlations.

Sets the variance (diagonal element) at position (idx, idx) to the given value and clears all off-diagonal correlations involving this state variable. This effectively makes the idx-th state variable uncorrelated with others.

Parameters
[in]idxZero-based index of the state variable (0 <= idx < Size_)
[in]valNew variance value (must be positive)
Note
This operation modifies both the diagonal and off-diagonal elements
Commonly used to reinitialize uncertainty for specific state components
